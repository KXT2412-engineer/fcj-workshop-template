### Internship Journey Summary

After the internship and efforts at First Cloud Journey, I have achieved significant milestones in both mindset and technical skills:

- **Mastering the AWS Ecosystem:** Hands-on setup and secure deployment of core cloud services including IAM, VPC, EC2, S3, Parameter Store, and SNS, moving beyond purely coding on Localhost.
- **Robust Backend Development:** Built an API system using C#/.NET Core with a clear layered architecture (API, BLL, DAL). Fully handled the JWT authentication flow, authorization, and methodical Database management through Entity Framework Core (Code-First).
- **Integrating Artificial Intelligence & Algorithms:** Applied AI in invoice extraction (OCR) and financial consulting (Prompt Engineering). Embedded the Levenshtein algorithm to optimize smart data matching.
- **Automation & Performance Optimization:** Excellently utilized the Hangfire library to handle Background Jobs, smoothly solving the problem of automatically closing and renewing the Periodic Budget across months without causing system bottlenecks.
- **True Software Engineer Mindset:** Upgraded security awareness (Pre-signed URLs, hiding Secret Keys), prepared a professional deployment environment using Docker, and ensured system quality (QA) with Unit Testing before launch.
